# 🎞️ VHS Video Store

A beautiful retro 80s-style movie discovery app with neon VHS aesthetics, Firebase Auth, Firestore real-time sync, and TMDB integration.

![VHS Banner](public/vhs-banner.png) <!-- Add your own banner image later -->

## ✨ Features
- Neon magenta/cyan VHS theme with scanlines, grain & flicker effects
- Email + Google sign-in
- Search + trending movies (TMDB)
- Real-time favorites with live sync
- Movie details with posters & backdrops
- Protected routes & per-user Firestore data
- Fully responsive

## 🛠️ Tech Stack
- **Next.js 15** (App Router) + TypeScript
- **Tailwind CSS v4** + custom VHS design system
- **Firebase Auth + Firestore**
- **TMDB API**
- **Sonner** for toasts
